Setup Shopify CLI

Step 1 – Install Node.js

https://nodejs.org/en/download/
https://youtu.be/__7eOCxJyow

Step 2 – Install Ruby

https://rubyinstaller.org/downloads/

Step 3 – Install Git

https://git-scm.com/downloads

Step 4 – Install Shopify CLI

npm install -g @shopify/cli @shopify/theme

Step 5 – Check Version

shopify version

Error Fix:

https://stackoverflow.com/questions/56199111

Step 6 – Create Folder

mkdir folder-name

Step 7 – Open Folder

cd folder-name

Step 8 – Pull Theme

shopify theme pull -s

Step 9 – Logout

shopify auth logout

Step 10 – Dev Mode

shopify theme dev

Step 11 – Push Changes

shopify theme push
